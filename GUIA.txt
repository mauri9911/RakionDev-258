//--- INSTALACION:


1- Descargar el RakionPack
2- INstalar Navicat
3- INstalar 010_Editor
4- Instalar el Cliente RIS_Ver1026_XfsVer258.exe
5- Descargar Xampp
6- Instalar Xampp

NOTA:
No mostrare como se instalan estos programas ya que es muy simple, 
solo cubrire lo importante a partir del final de la instalacion del xampp y
abras el xampp-control panel

7- Abrir el Navicat y configurar los datos de conexion.

//--- CONFIGURACION (SERVER):

* BrokenServer

1- Entrar en BrokenServer/Settings
2- Abrir Settings.ini y cambiar la IP
3- Abrir GameServers.ini y cambiar la IP
4- Abrir el BrokenServer.exe

NOTA:
si no puedes abrir el BrokenServer.exe puede que necesites instalar
el Microsoft .Net Framework 4.0

* WorldServer

1- Abrir el Navicat.
2- Crear una Nueva base de datos.
3- Importar Base de Datos.
4- Entrar a la carpeta WorldServer y abrir wordlserver.ini
5- configurar la IP, el acceso a MySQL y el usuario Administrador del Sistema

NOTA:
Es muy importante que tu sistema posea una contraseńa y en caso de no tenerla, 
aca les muestro como hacerlo...

6- Click Derecho en el Acceso Directo "Install Service" y luego click en propiedades
7- En la pestańa "Acceso directo" debes configurar la ubicacion del archivo "RakionWorldServ.exe"
8- Repetir el Paso anterior para Start, Stop y Uninstall

El WorldServer corre como servicio de sistema, por lo tanto debe instalarse el servicio
antes de poder ejecutarlo! y para eso se debe hacer lo Siguiente:

Click derecho en "Install Service" y ejecutar como Administrador.
(Se instalará el servicio)

Click derecho en "Start Service" y ejecutar como Administrador.
(Se ejecutará el RakionWorldServ.exe)

En caso de que desees detener el servidor, debes usar "Stop Service" y en caso de querer desinstalar
el servicio, debes ejecutar "Uninstall Service"



** Completado la configuracion del servidor.




//--- CONFIGURACION (CLIENTE):


1- Copiar el archivo rakion.bin y load.bin dentro de la carpeta Patch/(EN/ES) 
a la carpeta "Bin" de donde tengas instalado el rakion.

2- El cliente necesita ser ejecutado por el Load.bin (que es un launcher especial 
que ayuda a traspasar el GameGuard) por lo tanto el launcher original de softnyx 
(NyxLauncher.exe) debe editarse para que abra el load.bin en vez del rakion.bin
y para ello, debemos abrir el 010 Editor y hacer lo siguiente...

3- Copiar los archivos ubicados en la carpeta WEB del pack a la carpeta htdocs del xampp

4- Cambiar el Acceso a MySQL en config.php y fetch.php

5- El launcher necesita verificar si hay nuevas actualizaciones o no, y para ello
necesitamos configurar el fetch.php

NOTA:
editar el Nyxlauncher.ini es muy simple de modificar y a la vez innecesario 
para este tutorial y lo pasaremos por alto.

6- En la carpeta Tools, abre el iXFS.exe y abre NyxLauncherEnc.xfs para editar la ruta
de nuestro fetch.php y el puerto de nuestro BrokenServer

NOTA:
el iXFS editor tiene un pequeńo truco para extraer y leer los datos 
(que hasta ahora no tengo idea de si fue hecho asi a proposito o no)

7- El load.bin ademas de abrir el juego, este primero se comunica con sus archivos web
para saber si debe o no, revisar que los archivos importantes del juego no hayan sido 
modificados y para ello debemos configurar los archivos config.php y file.php
(que por ahora no explicare como se configuran y dejaremos todo como esta)

Para que se entienda mejor que archivo depende de que programa:

Nyxlauncher.exe -> fetch.php
Load.bin -> file.php, launcherlogin.php, config.php

8- Ahora el Load.bin debe saber a donde se conectara para buscar los archivos y para 
ello se necesita generar un archivo que contenga esa informacion...
ir a la carpeta Tools y abrir GConfig.exe y escribir la ruta de tu sitio web 
donde esten ubicado los archivos (que para este ejemplo seria localhost)

9- Mover el archivo generado "config.xfs" a la carpeta Bin dentro de la carpeta Rakion

10- Hasta ahora todo lo relacionado con el launcher ha sido finalizado y solo nos queda
configurar los datos de nuestro servidor en la data del juego y para ello abriremos la
herramienta iXFS.exe y editaremos el Datasetup.xfs de Rakion con los datos de nuestro 
servidor.

11- Abrir el Navicat y Crear un usuario en la tabla "user" para poder iniciar sesion.

12- Ejecutar el Nyxlauncher.exe y a jugar.

NOTA:
el Nyxlauncher no verifica que el usuario y contraseńa que ingresaste sean validos,
por lo tanto si escribes mal los datos igual abrirá el juego pero no podras entrar.


Y con esto finalizamos este tutorial de Como crear tu servidor de Rakion v258.
