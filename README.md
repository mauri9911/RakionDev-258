## 🚀 Estructura del Proyecto

El repositorio está dividido en tres pilares principales para facilitar el desarrollo modular:

| Carpeta | Componente | Tecnología | Descripción |
| :--- | :--- | :--- | :--- |
| `📂 /client` | **Cliente** | [Serious Engine] | Todo lo relacionado con el arte, UI y mecánicas del jugador. |
| `📂 /server` | **Server Files** | [C++] | Lógica de red, base de datos y validaciones de seguridad. |
| `📂 /web` | **Plataforma Web** | [PHP / HTML / CSS] | Landing page, ranking de jugadores y panel de administración. |
