////////////////////////////////////////////////////////////////////////////////////
// ---------------------- Partly Functional Characteristics ----------------------//
////////////////////////////////////////////////////////////////////////////////////

* Refinery:
- Upgrading an item only increases their statistics to "+1" for each "+1"
- Upgrading an item does not bring their shining color that everyone knows.
(because this client is from the age where those distinctive colors didn't exist.)

* Guilds:
- Guilds doesn't have a "specific purpose" rather than just give a yellow color to 
your name...
- There is no way to visualize your members from the game menu, add or expel them...
- There is no way to organize guild wars and get points from that.
- For some strange reason, you can view your guild points and your position.

NOTE: To do these two things, you have to edit directly the database.


////////////////////////////////////////////////////////////////////////////////////
// ---------------------- Know Errors in the Game (SERVER) -----------------------//
////////////////////////////////////////////////////////////////////////////////////

* Messenger:
- It doesn't opens! and although you can open it with the command, whatever you type 
will not reach to anybody.

* Refinery
- If you try to upgrade some item, WorldServer will freeze.

////////////////////////////////////////////////////////////////////////////////////
// ---------------------- Know Errors in the Game (CLIENT) -----------------------//
////////////////////////////////////////////////////////////////////////////////////

* Disconnections/Closing without notice:
- This cause is unknown... but it happens when there are quite sounds/graphics effects 
in some spot and it happens a lot when you enter in a battle against many players 
with creatures or stages full of creatures and effects like lava maps.
